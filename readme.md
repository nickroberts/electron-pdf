# electron-pdf

> My cool app


## Dev

```
$ npm install
```

### Run

```
$ npm start
```

### Rebuild modules for Electron

```
$ npm run rebuild
```

_Note: you might need to run this on Windows:_

```
$ npm run rebuild:im
```

### Build

```
$ npm run build
```

Builds the app for macOS, Linux, and Windows, using [electron-packager](https://github.com/electron-userland/electron-packager).
